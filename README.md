## Sobre o Projeto

Projeto desenvolvido como atividade da disciplina de Desenvolvimento Web do curso de Engenharia da Computação (CEUB).

A proposta consistiu na criação de um portfólio digital para um fotógrafo profissional fictício, com foco na apresentação visual de trabalhos fotográficos e na experiência do usuário.

O projeto explora conceitos modernos de desenvolvimento front-end, incluindo galerias interativas, navegação dinâmica, animações, efeitos visuais avançados e formulários personalizados. Toda a estrutura foi desenvolvida utilizando HTML, CSS e JavaScript puro, priorizando desempenho, responsividade e usabilidade.

Além da apresentação dos ensaios fotográficos, a aplicação oferece mecanismos de interação que aproximam o visitante do conteúdo, criando uma experiência semelhante à encontrada em portfólios profissionais do mercado.

## Funcionalidades

* Navegação fixa durante toda a navegação
* Sidebar com atalhos rápidos para as seções
* Scroll suave entre os conteúdos
* Galerias fotográficas organizadas por categoria
* Navegação horizontal por arrasto (drag to scroll)
* Compatibilidade com dispositivos móveis
* Zoom interativo em imagens
* Overlay para visualização ampliada
* Formulário de contato personalizado
* Simulação de envio de mensagens
* Destaque automático da seção ativa
* Elementos gráficos animados em segundo plano
* Layout totalmente responsivo
